自动化打包测试之Fir.im

前言

目标：运行sh脚本，一键打包上传测试。

操作步骤

一、配置脚本环境

1、安装 rvm

curl -L get.rvm.io | bash -s stable

source ~/.bashrc

source ~/.bash_profile

rvm -v

2、安装 fir-cli 套件

gem install fir-cli

二、安装脚本

1、下载脚本，地址

2、将以下3个文件拖到项目根目录（XXX.xcodeproj文件同级）

exportTest.plist

exportAppstore.plist

shell.sh

3、更新.plist文件

两个.plist文件中添加一个字典

<key>provisioningProfiles</key>
    <dict>
        <key>com.lianjia.name</key>
        <string>{iOS Provisioning Profiles Name}</string>
    </dict>

4、替换打包类型

特别说明：如果项目没有采用Cocoapods管理，没有.xcworkspace,只有xcodeproj；需要将脚本中这句

//cocoapods 管理项目
xcodebuild \
archive -workspace ${project_path}/${project_name}.xcworkspace \

替换成下面这句

//没有cocoaPod项目
xcodebuild \
archive -project ${project_path}/${project_name}.xcodeproj \

5、修改项目名字

6、修改fir平台Token

三、执行脚本

1、cd到项目根目录

2、终端输入

./shell.sh

成功！
