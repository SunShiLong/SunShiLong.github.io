ActiveSQLite数据库操作说明

2018-05-07

ActiveSQLite数据库操作说明

前言

pod "ActiveSQLite"

优点：

  1. 自动建表
  2. 基于SQLite.swift数据库
  3. 不需要写Expression表达式
  4. 增删改查支持String和key-value字典操作
  5. 查询结果自动包装成对象
  6. 支持事务操作和异步操作
  7. 模型和数据库表映射
  8. 带有日志系统
  9. 数据库可以升级

一、特点

1、支持的数据类型

NSNumber类型对应SQLite.swift的两种类型（Int64和Double)。NSNumber默认的映射类型是Int64。重写DBModel的doubleTypes()方法能标记属性为Double类型。

class T_BLE_DEVICE: DBModel {

    var Major:NSNumber?
    var Minor:NSNumber?

    var RssiAnd: NSNumber?
    var RssiIos: NSNumber?
    var RssiDlj: NSNumber?
    var AccIos : NSNumber?
    var IsAdjusted:NSNumber?

    //标记double类型
    override func doubleTypes() -> [String]{
        return ["AccIos"]
    }
}

二、开始

1、手动建数据库

func createMyselfDB() {

        let fileManager = FileManager.default

        let myDirectory = NSHomeDirectory() + "/Documents/DBFile/"
        let filePath = myDirectory.appending("aaa.db")
        print(filePath)
        let exsit = fileManager.fileExists(atPath: filePath)

        if !exsit {
            //创建数据库
            try! fileManager.createDirectory(atPath: myDirectory, withIntermediateDirectories: true, attributes: nil)

            let result = fileManager.createFile(atPath: filePath, contents: nil, attributes: nil)

            print(result)
        }
}

1.1连接现有数据库

//连接现有数据库
        let dbFile = Bundle.main.path(forResource: "DataBase", ofType: "db")

2、手动建表

MARK: ---创建表（可以省略，数据库会自动创建表）
        do {

2.1、自动建表

为了演示数据库操作建立两个模型Person.swift和T_BLE_DEVICE.swift

//model1
import UIKit
import ActiveSQLite

class Person: DBModel {
    var name:String!
    var price:NSNumber!

    var age: NSNumber!

}


//model2
import UIKit
import ActiveSQLite

class T_BLE_DEVICE: DBModel {

    var Major:NSNumber?
    var Minor:NSNumber?

    var RssiAnd: NSNumber?
    var RssiIos: NSNumber?
    var RssiDlj: NSNumber?
    var AccIos : NSNumber?
    var IsAdjusted:NSNumber?

    //标记double类型
    override func doubleTypes() -> [String]{
        return ["AccIos"]
    }
}

手动建表这步可省略，数据库会根据模型自动建表

//MARK: ----- 连接数据库

三、CURD（增改查删）

===================================== 增 ============================================

1.1、保存一条数据

//MARK: --- 1、保存数据
                let per:Person = Person()
                per.name = "Alon"
                per.price = 13
                per.age = 5
                try! per.save()

1.2、插入一条数据

//MARK: --- 2、插入一条数据
                let person1 = Person()
                person1.name = "Bolee"
                person1.price = 25
                person1.age = 5
                try! person1.insert()

1.3、插入多条数据

//MARK: --- 3、插入多条数据
                var personArray = [Person]()
                for i in 1...8 {
                    let p = Person()
                    p.name = "student-\(i)"
                    p.price = 10 + i as NSNumber
                    p.age = 5
                    personArray.append(p)
                }
                try! Person.insertBatch(models: personArray)

===================================== 查 ============================================

2.1、查询第一条数据

let person = Person.findFirst("name", value: "Bolee") as! Person
print(person)

2.2、查询多条数据

let persons = Person.findAll(["name" : "Bolee"], ["price" : true]) as! [Person]
print(persons.count)

2.3、通过Expression查询

let id = Expression<NSNumber>("id")
let name1 = Expression<String>("name")
let price = Expression<NSNumber>("price")
let array = Person.findAll(name1 == "Bolee") as! Array<Person>
print(array.count)

let array1 = Person.findAll(id < NSNumber(value: 10), orders: [Person.id.asc]) as! Array<Person>
print(array1.count)

2.4、链式查询

let array3 = Person().where(Expression<NSNumber>("id") > 3)
    .order(Person.id)
    .limit(20)
    .run() as! [Person]

print(array3.count)

===================================== 改 ============================================

3.1、通过属性值修改

person.name = "Angle"
try! person.save()
print(person)

3.2、通过属性名字符串和属性值

try! person.update("price", value: NSNumber(value: 100))       //单属性修改
try! person.update(["name": "Dove","price":NSNumber(value: 200)]) //多属性修改

try! Person.update(["name": "Maly","price":NSNumber(value: 300)], where: ["id": NSNumber(value:1)]) //批量修改

3.3、通过SQLite.swift的Setter

let price = Expression<NSNumber>("price")
try! person?.update(price <- NSNumber(value: 110))

===================================== 删 ============================================

4.1、删除一条数据

try? person?.delete()

4.2、清空表

try? Person.deleteAll()

4.3、链式删除

try? Person().where(Expression<NSNumber>("id") > 20)
            .runDelete()

四、高级用法

1、表重命名、

try? Person.renameTable(oldName: "Person", newName: "Pepole")

2、添加新的属性

  • 1、先在模型类里添加属性

  • 2、调用addColumn

    try? Person.addColumn(["age"])   //可添加多个属性

3、删除表

try? Person.dropTable()

4、异步事务（重点来了….）

  • 建议把所有的insert，update，delete操作和alter表的代码全部放在ActiveSQLite.save代码块中。一个块中的sql操作在同一个事务当中
  • 建议所有增删改查操作用异步事务saveAsync
func makeAsync() {
        ActiveSQLite.saveAsync({

            var ps = [Person]()
            for i in 0 ..< 3 {
                let p = Person()
                p.name = "Teacher-\(i)"
                p.price = i as NSNumber
                ps.append(p)
            }
            try! Person.insertBatch(models: ps)

        }, completion: { (error) in

            if error != nil {
                debugPrint("transtion fails \(error.debugDescription)")
            }else{
                debugPrint("transtion success")
            }

        })
    }
